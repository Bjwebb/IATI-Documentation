Example Usage
~~~~~~~~~~~~~

The budget element acts as a container for budget details.

At this level, the attribute “type” should be used to declare whether
the budget is an Original (code 1) or Revised (code 2). If omitted
Original is assumed:

.. code-block:: xml

        <budget type="1">
        ....
        </budget>
