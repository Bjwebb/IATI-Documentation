Example Usage
~~~~~~~~~~~~~

The budget element acts as a container for transaction details.

At this level, the attribute “ref” can be used to describe reference to
this transaction in another system:

.. code-block:: xml

        <transaction ref="200304678">
        ....
        </transaction>
