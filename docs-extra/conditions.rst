Example Usage
^^^^^^^^^^^^^

This is a container element for conditions. Declared through the
following yes/no (1/0) value stating whether there are conditions
attached to the activity.

For no conditions: ``
<conditions attached="0" />
``

For conditions:

``
<conditions attached="1">
....
</conditions>
``
